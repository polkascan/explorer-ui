Polkascan

Announcement

As of December 31st 2023, the hosted instance of our open source block-explorer for the Polkadot and Kusama ecosystem is retired.

This instance relied heavily on a publicly available datasource, which has been announced to sunset in the near future.

Since our last treasury proposal was downvoted, we see no immediate way forward to fund further development of the Polkascan explorer. The latest version of the source code will, of course, still be available at GitHub.

View on GitHub

We want to thank all the people we were involved with in this project over the past five years. It was inspiring to be a part of this community.

The Polkascan Team
